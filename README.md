# r307 Fingerprint Module on ESP32
The Following Code is a proper Library for interfacing r307 Fingerprint Module with ESP32 on Embedded C Language

# GPIO Functions:
| ESP32 Devkit v1   | r307 Fingerprint Sensor |
| ------------- | ------------- |
| VIN pin | Pin 1 |
| GND pin | Pin 2 |
| GPIO 16 | Pin 3 |
| GPIO 17 | Pin 4 |

* Pin 5 and Pin 6 of the sensor can be opened ( No Connection )
